🏠 B2S KPI 시스템이란?
B2S 컴퍼니 전 부서의 KPI(핵심성과지표)를 실시간으로 입력·관리·공유하는 시스템입니다. 각 담당자가 직접 실적을 입력하면 전체 대시보드에 자동 반영됩니다.
🔑 로그인 방법
1
우측 상단 "Google로 로그인" 버튼 클릭
2
본인 구글 계정으로 로그인
3
로그인 후 본인 부서 탭으로 이동하여 업무 시작
💡 로그인하면 내가 입력한 데이터가 팀 전체에 실시간으로 공유됩니다
👥 부서별 담당자
${Object.entries(DEPT_OWNERS).map(([dept,name])=>`
${DEPTS[dept]?.icon||'👤'}
${name}
${dept}
`).join('')}
📱 앱 설치 방법 (PWA)
A

PC: 주소창 오른쪽 설치 아이콘 클릭
💡 설치 후 앱처럼 사용 가능. 오프라인에서도 기존 데이터 조회 가능합니다

💡 입력 즉시 달성률 미리보기가 표시됩니다. 저장 전 확인하세요

✅ 주간 액션 작성 방법
1
"실적 입력" 탭 → ➕ 액션 추가 탭 클릭
2
부서 KPI 참고 패널에서 부서 선택 → KPI 목록 확인
3
관련 KPI 항목 탭하여 자동 입력
4
실행 내용 입력, ＋ 실행 추가로 복수 실행 추가 가능
5
결과·발생 문제·다음 조치 입력 후 ➕ 액션 저장
💡 저장된 액션은 "주간 점검" 탭에서 체크(완료) 처리할 수 있습니다
✏️ 작성 원칙
관련 KPI: 이번 주 실행이 어떤 KPI와 연관되는지 명시 (예: 월 거래처 방문 건수)

실행 내용: 구체적으로 작성 (예: "거래처 방문" ❌ → "A식품 담당자 미팅 및 신제품 샘플 전달" ✅)

결과: 수치로 기록 (예: "잘 됨" ❌ → "샘플 2건 요청, 다음주 발주 예정" ✅)

발생 문제: 없으면 "없음" 기재. 문제 발생 시 구체적 원인 작성

다음 조치: 다음 주에 할 구체적 행동 작성
